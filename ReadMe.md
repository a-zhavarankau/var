##Varta 

###Description
It's a startup project focused on art. Developer task is to parse the defined sites ("https://kalektar.org/") to get the following information:
- Author name, occupation
- Author's events (name, dates, place, links)
<br><br>

###Instrunemts:
- asyncio, aiohttp
- multiprocessing
- requests
- requests_html
- JavaScript
- Burp Suite
